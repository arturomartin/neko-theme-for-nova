Neko theme for Nova

A minimal dark theme for Nova, the new editor by Panic. I created for myself and use it for my daily work. Use it as you like!

Inspired by the beautiful Ayu and Pure themes.

Installation

  • On Nova, go to Extensions > Extensions Library > Themes and install Neko from there.
  • After that, go to Nova > Preferences > Theme and choose Neko.
